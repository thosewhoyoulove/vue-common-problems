### 解决方案：将 Nginx 配置文件修改为下列代码

```Nginx

        server_name sciencewithfun;
        location /music {
            root  /www/server/phpmyadmin/music;
            index index.html index.htm index.php;
            try_files $uri $uri/ /music/index.html;
        }
```
